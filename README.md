# SceneAtlas

SceneAtlas is a cinematic research SaaS for movie search, AI insight generation, and personal film organization.

## Phase 1

- Next.js web app
- NestJS API
- PostgreSQL via Neon
- Prisma schema
- Public movie-data baseline using Wikidata and Wikimedia Commons
- Expensive-looking editorial UI for premium feel

## Local Setup

1. Copy `.env.example` to `.env` and fill in your environment values.
2. Install dependencies with `pnpm install`.
3. Run the web and API together with `pnpm dev`.

## Deployment Targets

- Web: Vercel
- API: Render
- Database: Neon
